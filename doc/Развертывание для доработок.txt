Инструкция тестировалась на Ubuntu 20.04

Желательно иметь не менее 3ГБ оперативной памяти

sudo apt install git npm
git clone https://github.com/keremet/satushem

Заменить ссылку на API в файле satushem/front/src/environments/environment.ts на свою.

Для скачивания всех зависимостей в каталоге satushem/front выполнить
npm install

Для запуска фронтовой части в каталоге satushem/front выполнить
npm start


Настройку сервера БД и веб-сервера смотреть в "Развертывание на VPS.txt". Модуль rewrite можно не настраивать, так как потребуются только каталоги api и img.
Веб-сервер удобно сделать на машине разработчика и при этом каталог api как символическую ссылку на каталог в репозитории, чтобы не было необходимости копировать файлы в каталог сервера после каждого их изменения
